Conceito

A lógica dos grafos

Um grafo é uma estrutura de dois elementos: nós (as ideias) e arestas (as conexões entre elas). É diferente de pastas — não há hierarquia, só relações.

◉

Nós — as ideias

Cada nó é uma ideia atômica: um conceito, uma decisão, um insight. Uma nota = uma coisa só. Se cobre dois assuntos, são dois nós.

⟷

Arestas — as conexões

Cada aresta liga dois nós e explica por que se conectam — o mecanismo compartilhado. Sem o "porquê", a conexão não serve. É o que diferencia um grafo de uma lista de links aleatórios.

✦

Cross-domain — o valor real

Numa pasta, uma ideia de vendas fica com vendas. Num grafo, ela pode se conectar com psicologia cognitiva, produto e liderança ao mesmo tempo. É aí que o insight inesperado aparece.

Exemplo: uma nota sobre "urgência em vendas" conectada a "urgência em design de produto" via same_mechanism_as — "ambas exploram aversão à perda para acelerar decisão". Esse link só existe porque alguém (ou uma IA) percebeu o mecanismo compartilhado.

cross-domain — conexão entre áreas de conhecimento diferentes (ex: vendas ↔ produto)

Comparação

Por que não o Obsidian?

Obsidian é excelente — mas foi feito para humanos lerem notas. Expert Brain foi feito para a IA usar o conhecimento.

Obsidian

Você anota, você lê

É uma ferramenta de captura para humanos. A IA não acessa o vault automaticamente — você precisa colar o conteúdo na conversa.

Busca por palavra-chave

A busca é textual. Não entende semântica — "crescimento" não encontra notas sobre "expansão de mercado".

Links manuais

Você cria as conexões entre notas. Depende de você perceber o padrão — a IA não sugere o link inesperado.

Expert Brain

A IA salva e busca sozinha

Você conversa normalmente. O Claude decide o que vale guardar, atomiza e salva. Na próxima sessão, consulta antes de responder.

Busca semântica (bge-m3)

Embeddings multilíngues. "crescimento" encontra notas sobre "expansão", "escala", "tração" — em português, inglês ou o que for.

Edges cross-domain automáticos

O vault liga uma ideia de gestão com um padrão de produto. A conexão inesperada aparece — é aí que o insight mora.

bge-m3 — modelo de IA especializado em busca semântica em mais de 100 idiomas
edges — arestas: as conexões entre nós no grafo, cada uma com um "porquê" explicado
cross-domain — conexão entre áreas de conhecimento diferentes
Compatibilidade

Funciona em qualquer ferramenta com MCP

MCP é um protocolo aberto. Qualquer ferramenta de IA que o suporte pode usar o Expert Brain — não é exclusivo do Claude.

Recomendado

Claude Code

Via claude mcp add. Conexão permanente, ativa em toda sessão automaticamente.

Desktop

Claude Desktop

Settings → Connectors → Add custom connector. Cole a URL do vault.

Web

Claude Web

claude.ai → Settings → Connectors. Funciona direto no browser, sem instalar nada.

Qualquer MCP

Outros agentes

ChatGPT, Cursor, Windsurf, ou qualquer ferramenta que suporte o protocolo MCP pode se conectar.

MCP (Model Context Protocol) — protocolo aberto que permite ferramentas de IA se conectarem a serviços externos. Criado pela Anthropic, adotado pela indústria.
