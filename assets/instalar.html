Conceito

A lógica dos grafos

Um grafo é uma estrutura de dois elementos: nós (as ideias) e arestas (as conexões entre elas). É diferente de pastas — não há hierarquia, só relações.

◉

Nós — as ideias

Cada nó é uma ideia atômica: um conceito, uma decisão, um insight. Uma nota = uma coisa só. Se cobre dois assuntos, são dois nós.

⟷

Arestas — as conexões

Cada aresta liga dois nós e explica por que se conectam — o mecanismo compartilhado. Sem o "porquê", a conexão não serve. É o que diferencia um grafo de uma lista de links aleatórios.

✦

Cross-domain — o valor real

Numa pasta, uma ideia de vendas fica com vendas. Num grafo, ela pode se conectar com psicologia cognitiva, produto e liderança ao mesmo tempo. É aí que o insight inesperado aparece.

Exemplo: uma nota sobre "urgência em vendas" conectada a "urgência em design de produto" via same_mechanism_as — "ambas exploram aversão à perda para acelerar decisão". Esse link só existe porque alguém (ou uma IA) percebeu o mecanismo compartilhado.

Contexto

A lógica é a mesma do Obsidian

Notas atômicas, conexões com significado, grafo de conhecimento — Expert Brain não inventou nada. É a mesma filosofia do Obsidian e do Zettelkasten. A diferença é para quem foi construído.

Obsidian — feito para humanos
Você escreve as notas. Você cria os links. Você percebe os padrões. A ferramenta organiza o que você capturou.
Funciona muito bem — mas exige disciplina constante. A maioria das pessoas começa animada e abandona em semanas.
Com o Smart Connections, a IA ajuda a encontrar o que você já escreveu. Mas você ainda precisa escrever.
Expert Brain — feito para IA usar
O Claude escreve as notas durante a conversa. Você não precisa mudar de app nem lembrar de capturar.
O vault cresce sozinho à medida que você usa o Claude — sem disciplina adicional da sua parte.
O Claude também consulta o vault antes de responder. Não é só armazenamento — é memória ativa dentro do fluxo.
Tem Obsidian há anos? Não precisa abandonar. São ferramentas com propósitos diferentes — um organiza o que você pensa, o outro lembra pelo Claude o que você já discutiu.
Zettelkasten — método de notas atômicas criado por Niklas Luhmann, base filosófica do Obsidian e do Expert Brain
Smart Connections — plugin do Obsidian que adiciona busca semântica via OpenAI
Compatibilidade

Funciona em qualquer ferramenta com MCP

MCP é um protocolo aberto. Qualquer ferramenta de IA que o suporte pode usar o Expert Brain — não é exclusivo do Claude.

Recomendado

Claude Code

Via claude mcp add. Conexão permanente, ativa em toda sessão automaticamente.

Desktop

Claude Desktop

Settings → Connectors → Add custom connector. Cole a URL do vault.

Web

Claude Web

claude.ai → Settings → Connectors. Funciona direto no browser, sem instalar nada.

Qualquer MCP

Outros agentes

ChatGPT, Cursor, Windsurf, ou qualquer ferramenta que suporte o protocolo MCP pode se conectar.

MCP (Model Context Protocol) — protocolo aberto que permite ferramentas de IA se conectarem a serviços externos. Criado pela Anthropic, adotado pela indústria.
Instalação

2 passos — o Claude faz o resto

1

Crie uma conta na Cloudflare

Só email — sem cartão de crédito. Acesse dash.cloudflare.com/sign-up, crie a conta e confirme o email.

2

Abra o Claude Code em qualquer pasta e cole esse prompt:

instala o Expert Brain do repo Expert-Integrado/expert-brain
Um browser vai abrir pedindo permissão na Cloudflare — clique Allow. Depois informe seu email e senha do vault.
Guarde a senha. Não tem recuperação.
